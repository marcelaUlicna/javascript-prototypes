Class: SpinnerEvents

SpinnerEvents

Constructor

new SpinnerEvents(spinner, direction, value, startDown)

Base class that implements IEvents interface. It provides basic implementation of all events. It contains virtual method "changeValue" which is overridden in each inherited class.
Parameters:
Name Type Description
spinner Spinner Class that contains all necessary properties
direction string Value "up" or "down" defines increasing or decreasing value
value string Input value
startDown boolean Private property that identifies whether mouse down is holding
